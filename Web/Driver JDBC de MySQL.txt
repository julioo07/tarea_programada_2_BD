1. cd ~
2. wget https://dev.mysql.com/get/Downloads/Connector-J/mysql-connector-j-9.0.0.zip
3. unzip mysql-connector-j-9.0.0.zip

4. mkdir -p ~/music-loader/libs

5. cp mysql-connector-j-9.0.0/mysql-connector-j-9.0.0.jar ~/music-loader/libs/


Instalar MySQL

sudo apt install mysql-server -y
sudo service mysql start

Cargar Esquema en MySQL
sudo mysql < ~/music-loader/music_analysis_schema.sql
